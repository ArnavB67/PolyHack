POLYHACK is a fast paced arcade survival game set within a simulated terminal environment. You play as a defensive system node tasked with purging hostile geometric malware using rapid fire projectiles and a shockwave. As you take damage, your system's Hack Percentage rises, triggering intense screen-tearing glitch Effects, camera shake. The Goal is to purge as many malwares you can

Rapid Fire bullets have a cooldown of 0.5 seconds

Shockwave has a cooldown of 5 seconds and while the shockwave is active the players movement halts.
